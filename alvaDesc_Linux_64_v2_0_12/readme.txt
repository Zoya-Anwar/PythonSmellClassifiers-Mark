# alvaDesc

alvaDesc is the next generation tool for the calculation of a wide range of molecular descriptors and a number of molecular fingerprints.

## Installation

1) Close previous version of the program 
2) Unpack the tar.gz file, e.g.:
      tar -xf alvaDesc_Linux_64_v*.tgz -C /
3) Run the program:
      alvaDescGUI
